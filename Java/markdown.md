# 글씨 강조

1. 굵게 (Bold)<br>
   **Bold**<br>
   (** 내용 **) : 이런 식으로 굵게 표시할 수 있다.

2. 기울임 (Italic) <br>
   *Itatlic* <br>
   (* 내용 *) : 이런식으로 기울임 표시할 수 있다.

3. 취소선 (Strike)<br>
  ~~Strike~~<br>
  (~~ 내용 ~~) : 이런식으로 취소선을 표시할 수 있다.
