# refactoring
기존의 코드 기능은 유지하면서 내부 구조를 개선하여 가독성을 높이고, 유지보수를 용이하게 하는 과정이다.

쉽게 이야기해서 작동하는 기능은 똑같은데 코드를 개선하는 것.

# 배열생성과 초기화
```java

```

